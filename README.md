# Недостижимый реализм

> Реалистичная сборка Minecraft 1.20.1, целью которой является воссоздать человеческую историю начиная с каменного века, и заканчивая нашим временем. Финал - полёт в космос, такой пустой и безлюдный... Ванильные механики пределанны, чтобы соответствовать реализму, а промышленность, металлургия и космос реализованный через TerraFirmaCraft, Create и Immersive Engineering.

## Важно перед запуском

### Разблокировка рецептов ракетных двигателей

В сборке присутствует мод **Creating Space**, содержащий десятки чертежей ракетных двигателей. Чтобы **сразу открыть все доступные дизайны**, используйте команду:
/creatingspace addAllDesigns

### Режим читов обязателен

> [!WARNING]
> **В одиночной игре мир должен быть открыт с включёнными читами!**
> 
> Команда `/creatingspace addAllDesigns` требует уровень прав оператора (OP) и **не работает без читов**.
